# E-handel-Webshop

Ett grupprojekt inom kursen *Grundläggande Programmering (50 YP)* på Teknikhögskolan.

Gruppmedlemmar: 
- Martin Friis (@MarFri8)
- Jose Muñoz (@JoseMunozO)
- Nils Paulsson (@nils-fjp)

Syftet är att bygga en responsiv e-handelsapplikation med HTML, CSS och JavaScript där användaren kan genomföra en komplett köpresa.
